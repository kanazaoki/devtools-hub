## 作ったもの

**Markdown Table Generator** — GUI でMarkdownテーブルを作成・列/行追加・配置指定・CSVインポート対応

https://devtools-hub.vercel.app/tools/markdown-table-generator

![Markdown Table GeneratorのスクリーンショットF](スクショURL)

## できること

- セルをクリックして直接テキスト編集
- 列・行の追加と削除（最低1列・1行を保持）
- 各列の配置（左/中央/右）を切り替えると区切り行（`:---` / `:---:` / `---:`）に即反映
- CSV をペーストして一括インポート（1行目をヘッダーとして取り込み）
- Markdown 出力のリアルタイムプレビューとワンクリックコピー

## なぜ作ったか

Markdown のテーブル記法は列が増えるとパイプの位置を手で揃えるのが面倒です。特に列の追加や削除のたびに区切り行の `:---` を書き直すのが地味にストレスでした。CSV でデータを持っているケースも多いのでインポート機能も入れています。
